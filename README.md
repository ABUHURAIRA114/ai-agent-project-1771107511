# Personal Website
A simple website showcasing personal projects and skills

## Introduction
This repository contains the code for a personal website, built using HTML, CSS, JavaScript, and React. The website is designed to showcase personal projects and skills.

## Features
* Simple and responsive design
* Showcase of personal projects
* Display of technical skills
* Interactive elements using JavaScript and React

## Tech Stack
* HTML: Structuring and content creation
* CSS: Styling and layout
* JavaScript: Dynamic functionality
* React: Component-based UI

## File Structure
* `index.html`: Main entry point of the website
* `styles.css`: Global styles and layout
* `script.js`: JavaScript functionality
* `components`: React components for the website
	+ `Project.jsx`: Component for showcasing personal projects
	+ `Skills.jsx`: Component for displaying technical skills

## Getting Started
1. Clone the repository: `git clone https://github.com/your-username/personal-website.git`
2. Navigate to the repository: `cd personal-website`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open the website in your browser: `http://localhost:3000`

## Deployment
To deploy the website, simply upload the contents of the `public` folder to your hosting platform.
